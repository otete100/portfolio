SKILL

コーディング

HTML / CSS / Javascript等を用いてデザインを再現いたします。ユーザーが保守しやすいコーディングを心がけます。WordPressにおいては既存テーマを用いて投稿や固定ページの追加編集、プラグインの挿入が可能です。

使用ツール: VScode

WEBデザイン

WEBサイトデザインとバナー制作が出来ます。頻繁に更新が必要なサイトの場合は、WordPressをご提案します。掲載するテキストの内容が決まっていない場合についても、考えさせていただきます。

使用ツール: Photoshop / Illustrator
